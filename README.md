# 📊 Swiggy Sales Analysis Dashboard | Excel

## Overview

This project presents an interactive **Swiggy Sales Analytics Dashboard** developed using **Excel** to analyze sales performance, customer ordering behavior, regional distribution, and restaurant insights. The dashboard enables users to monitor key business metrics, identify sales trends, compare regional performance, and support data-driven decision-making through interactive visualizations.

## Business Challenge

Food delivery businesses generate large volumes of sales data across multiple cities, restaurants, and food categories. Without a centralized dashboard, it becomes difficult to monitor sales performance, evaluate customer preferences, identify top-performing locations, and track key business metrics for informed decision-making.

## Solution

Using **Excel**, sales data was transformed into an interactive dashboard featuring KPI cards, trend analysis, regional sales mapping, restaurant performance, city-wise analysis, quarterly sales comparison, and food type distribution. Interactive filters allow users to explore insights across months, states, categories, and food types.

## Key Features

• Interactive KPI cards displaying Total Sales, Orders, Average Order Value, Ratings, and Rating Count.

•Monthly and daily sales trend analysis.

•Regional sales visualization using an India map.

•Top 10 restaurant performance analysis.

•Quarterly sales, ratings, and order analysis.

•Food type distribution (Veg vs Non-Veg).

•Dynamic filtering by Month, State, Food Type, and Category.
